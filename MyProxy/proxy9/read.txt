Faça um programa de proxy de rede. O proxy funciona como uma ponte entre dois pares, recebe a conexão em uma determinada porta e faz a ponte com o destino;

5,0 - O aluno deve compreender e explicar o codigo fonte;
1,0 - Log em arquivo de todas as conexões ip/porta/data/horário;
1,0 - Bloqueio de IPs na conexão de entrada;
0,5 - Alerta no console quando algum IP bloqueado tentar conectar;
2,5 - O programa deve funcionar no laboratório;

ORDEM DE EXECUÇÃO:
Primeiro abra o arquivo conect.py no terminal com python 3
ex: python3 conect.py

Segundo abra o arquivo bridge.py no terminal com python 3
ex: python3 bridge.py

Terceiro abra o arquivo cliente.py no terminal com python 3
ex: python3 cliente.py
